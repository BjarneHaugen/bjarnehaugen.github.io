# GitHub Pages Project

This project is a simple GitHub Pages site that showcases various projects. The main landing page displays the NASA Photo of the Day and provides navigation to other pages, including a dedicated projects page.

## Project Structure

- **index.html**: The main landing page that displays the NASA Photo of the Day and includes navigation links to other pages.
- **projects.html**: A page that will list all the projects added to the site, complete with links and descriptions.
- **styles/styles.css**: The CSS file that styles the HTML elements for both the index and projects pages.
- **scripts/app.js**: The JavaScript file that handles dynamic content and interactivity for the project.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the project in your preferred code editor.
3. To view the site, open `index.html` in a web browser.
4. Modify `projects.html` to add new projects as needed.
5. Update `styles/styles.css` to change the appearance of the site.
6. Use `scripts/app.js` to add any interactive features or dynamic content.
